1. While
    - Odradjen je osnovni while iskaz.
    - argumenti u while-u mogu biti:
        - promenlijva
        - literal
        - relacioni izraz

    primeri:
        while(1){       // zauvek
            ...
        }

        while(x){       // Sve dok je x != 0
            ...
        }

        while(y < u){   // Sve dok je y manje od u
            ...
        }
